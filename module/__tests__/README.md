# Tests

Jest test suites for the Marvel Multiverse system live here.

- Place test files in this directory named `*.test.js`.
- Foundry VTT globals are mocked in `../__mocks__/` and loaded automatically
  via the `jest.setupFiles` configuration in `package.json`.
- Run the suite with `npm test`.

This directory currently contains the test framework only; no tests have been
written yet.
